## üêõ Troubleshooting

### Issue: Demo mode not activating
**Solution:**
1. Clear browser cache and cookies
2. Try incognito/private browsing mode
3. Manually set demo mode:
   ```javascript
   // Open browser console (F12)
   localStorage.setItem('recruitos_demo_mode', 'true');
   localStorage.setItem('recruitos_admin_mode', 'true');
   location.reload();
   ```

### Issue: "Sign up required" or payment prompts appearing
**Solution:**
- Verify demo mode is active (check URL has `?demo=true`)
- Press `Ctrl+Shift+A` to toggle admin mode
- Contact support if issue persists

### Issue: Slow loading or timeout errors
**Solution:**
- Refresh the page
- Check internet connection
- Try alternative URL: https://recruit20-git-master-arnarssons-projects.vercel.app?demo=true
- Vercel deployments may have cold starts (first load slower)

### Issue: Missing candidate data in demo
**Solution:**
- Click "Reset Demo Data" if available
- Reload page with `?demo=true` parameter
- Demo data should auto-populate on first visit
