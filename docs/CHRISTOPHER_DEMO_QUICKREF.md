## üö® Troubleshooting

**Demo mode not working?**
1. Add `?demo=true` to URL
2. Or press `Ctrl+Shift+A`
3. Or open browser console (F12) and run:
   ```javascript
   localStorage.setItem('recruitos_demo_mode', 'true');
   location.reload();
   ```

**Payment prompts showing?**
- Reload with `?demo=true` parameter
- Clear browser cache
- Try incognito mode
