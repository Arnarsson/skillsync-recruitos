## üõ†Ô∏è Troubleshooting

### Demo Link Not Working
**Symptoms:** 404 error, Vercel not found, blank page

**Solutions:**
1. Try alternative URL: `https://recruit20-git-master-arnarssons-projects.vercel.app?demo=true`
2. Check Vercel deployment status: `vercel ls --prod`
3. Redeploy if needed: `vercel --prod`
4. Use localhost for local demo: `npm run dev` then `http://localhost:3000?demo=true`

### Demo Mode Not Activating
**Symptoms:** Login prompt appears, payment gates shown, no sample data

**Solutions:**
1. Verify URL includes `?demo=true` parameter
2. Press `Ctrl+Shift+A` to manually enable admin mode
3. Open browser console (F12) and run:
   ```javascript
   localStorage.setItem('recruitos_demo_mode', 'true');
   localStorage.setItem('recruitos_admin_mode', 'true');
   location.reload();
   ```
4. Try incognito/private browsing mode
5. Clear browser cache and cookies

### Screen Recording Issues
**Symptoms:** Audio echoes, screen lags, file too large

**Solutions:**
- **Echo:** Record in smaller room, use directional microphone
- **Lag:** Close background apps, record at 1280x720 instead of 1080p
- **File size:** Use H.264 codec, compress with HandBrake, upload to Loom/YouTube
